Docker

docker:暂时可以理解为一个虚拟机，可以把node和npm 装在里面，

镜像->通过镜像变->容器、仓库

概念

  • docker是一个开源的应用容器引擎
  • 基于go语言实现
  • Docker可以让开发者打包他们的运用以及运用依赖的环境的（node mysql）到一个轻量级、可移植的容器中，
    然后发不到任何流行Linux机器上
  • 容器之间相互隔离，不受牵制
  • 容器性能开销极低

小结：Docker是一种容器技术，解决跨环境迁移问题
