    margin 穿透或重叠以及解决办法 （解决办法 BFC 块级格式化上下文）

    穿透发生在父子元素之间，子元素设置 margin 会影响其父元素
    重叠是同级元素在垂直方向上都设置了 margin

    解决办法：利用 BFC （首先变为 BFC 是一个 block 元素）

    BFC:特性
    1.BFC 中的元素不会影响到其之外的元素
    2.Box 垂直方向的的距离由 margin 决定，同一个 BOX 中同级元素垂直方向上 margin 会发生重叠
    3.Box 区域不会与 floatBox 发生重叠
    4.BFC 计算高度时，浮动元素的高度也参与计算

    BFC:触发条件

    1. float: left / right
    2. position: absolute / fixed
    3. overflow: auto / scroll / hidden
    4. display: inline-block / table-caption / table-cell

    移动端适配

    1.使用 media 媒体查询 2.使用 flex 弹性布局 3.使用 rem：rem 是根据 html 上的字体大小决定的 1rem=html font-size 的大小； 4.使用 vw : 100vw==屏幕宽度

    ####清除浮动

    1.给父元素添加高度/最后添加一个子元素：设置 clear：both/设置伪元素清除浮动/将父元素变为 BFC（块级格式上下文）：overflow:hidden

    css hock

    CSS hack 的原理
    由于不同的浏览器和浏览器各版本对 CSS 的支持及解析结果不一样，以及 CSS 优先级对浏览器展现效果的影响，我们可以据此针对不同的浏览器情景来应用不同的 CSS。
